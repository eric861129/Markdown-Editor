### 規格書 (PRD)

#### 專案概述：新手友善的互動式 Markdown 編輯器
* **目標核心**：打造一個「邊用邊學」的線上 Markdown 編輯器，透過互動式引導與任務機制，消除初學者的學習門檻。
* **部署環境**：純前端靜態網頁（Static Site），部署於 GitHub Pages。
* **主要產出**：支援一鍵匯出 `.md`、`.html`、`.pdf` 格式。

#### 核心功能一：沈浸式「初學者教學模式」設計
1.  **互動式引導 (Interactive Onboarding) / 闖關任務**
    * 進入編輯器時，預設載入一份「教學樣板文件」。
    * **任務導向**：左側編輯區會有提示框。當使用者正確輸入並觸發 Markdown 渲染時，給予視覺回饋（例如綠色勾勾）並解鎖下一個任務。
2.  **智慧側邊欄與樣板插入 (Smart Sidebar & Snippets) ✨ (新增)**
    * **雙模式切換**：編輯器左側設有專屬側邊欄，頂部提供「Markdown」與「Mermaid 圖表」切換按鈕。
    * **一鍵寫入**：依據當前模式，側邊欄會顯示對應的語法按鈕（如：粗體、清單，或是流程圖、圓餅圖）。點擊後，直接在游標處插入**帶有註解的樣板程式碼**，並自動選取可替換的文字，讓使用者直接從「改字」開始學習。
3.  **即時對照與溫和錯誤提示**
    * 標準的左右雙欄（左側編輯、右側預覽）。
    * **一般語法提示**：若語法不夠標準（如清單 `-` 後少空白），給予非干擾性的底線提示。

#### 核心功能二：進階 Mermaid 圖表支援與友善除錯 ✨ (新增)
1.  **即時圖表渲染**：支援 ````mermaid ```` 區塊，右側預覽區即時渲染 SVG。
2.  **友善錯誤攔截 (Graceful Error Handling)**：
    * 攔截 Mermaid 預設的解析錯誤。當使用者打錯圖表語法時，右側不要顯示可怕的錯誤代碼或破圖。
    * 替換為溫和的 UI 提示（例如：「圖表語法好像有一點小狀況喔！請檢查箭頭或括號。」），並嘗試指出錯誤發生的行數。
3.  **圖表闖關任務**：提供從修改文字到認識形狀、箭頭的分階教學。

#### 核心功能三：基礎編輯與匯出能力
1.  **純前端匯出功能**：支援 `.md` (Blob 下載)、`.html` (包裹基礎 CSS)、`.pdf` (需確保 Mermaid SVG 已完整渲染後再截取轉換)。
2.  **自動儲存**：利用 LocalStorage 自動暫存輸入內容。
